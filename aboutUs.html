About Me

Joaquim Crous

Sant Feliu de Llobregat

My name is Joaquim and I’m a person very committed to work, with the desire to improve myself and continue learning. I’m interested in a position that allows me to develop my potential, while increasing my skills and studies. I have finished a java backend web development bootcamp at IronHack, which has helped me to have a good level to develop restAPIs in Java using Spring framework connected to a MySQL database. In November 2022, I participated in the hackathon driven by Nuwe in the Jump2Digital22 convention. My team and I decided to make a web-app that allows the user to analyze the data of three types of supermarkets to improve their economic performance. We used restAPI in Java for the backend, React for the frontend and Python to make the stadistics. Also I have done a course at Udemy of Python where I learned to code different scripts as a simple manager. I learned by myself how to create a compose container using Docker to use my online bank API plus a MySQL database. Now I’m doing another bootcamp at IronHack of frontend where I would improve my skills in HTML, CSS and JS also implementing the VUE framework. You can see all my projects on my github repository: https://github.com/QuimCrous. In it you will find a bank restAPI connecting the Spring framework with a MySQL database, which is also in Docker format. You will also find other projects such as a CRM, management applications and my frontend projects as a clone of the Spotify website.
